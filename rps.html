How to Play

Rock Paper Scissors is a classic hand game played against the computer.

  • Click one of the three buttons: Rock, Paper, or Scissors.
  • The computer will randomly pick one of the three as well.
  • The winner is decided using the classic rules:
  • Rock beats Scissors
  • Scissors beats Paper
  • Paper beats Rock
  • If both players pick the same choice, it's a draw.
  • Your score, the computer's score, and the number of draws are tracked at the top.
  • Click "Reset Score" at any time to set all scores back to zero.

About This App

Rock Paper Scissors is a simple web application built using only:

  • HTML – for the page structure (header, sections, buttons)
  • CSS – for styling the layout, colors, and buttons
  • JavaScript – for generating the computer's random choice, comparing choices, deciding the winner, and updating the score

No external libraries or frameworks were used. All logic runs directly in the browser using plain JavaScript.
